# الحزمة النهائية للتسليم

هذا المجلد يحتوي النسخة المنظمة الجاهزة للنقل إلى Word أو للمراجعة النهائية.

## المحتويات

- `ACADEMIC_REPORT_AR.md`
  التقرير الأكاديمي النهائي، ويتضمن الصور مضمّنة داخل الملف نفسه عبر مسارات نسبية.

- `arrhenius_power_law_model.json`
  ملف النموذج الناتج، ويتضمن المعاملات، ونتائج التحقق، ومعلومات اختيار النموذج.

- `images/`
  مجلد الصور المستخدمة داخل التقرير.

## ملاحظة

عند فتح الملف:

```text
report_assets/final_package/ACADEMIC_REPORT_AR.md
```

يفترض أن تظهر الصور مباشرة لأن مساراتها أصبحت نسبية إلى مجلد `images`.
